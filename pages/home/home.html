Usage

angular.module('general-utilities-demo').controller('HomeCtrl', function ($scope, $rootScope, cskiwiUtilities) {
    $scope.networkInfo = cskiwiUtilities.networkInfo;
    $scope.deviceInfo = cskiwiUtilities.deviceInfo;
});
the variables are auto updated when the value changes

Events

  • onlineUpdate : when it's updated from online / offline
  • connectionTypeUpdate : when the netwerk type changed

Example output

Output for this device is: 
{{networkInfo|json}}
{{deviceInfo|json}}
